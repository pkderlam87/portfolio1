Footer

What was changed:

The image of the brand in the footer was replaced with an h1. The social media icons, now, are js functions to the corresponding social media. It has anchor tags to the products, about, and contact. The footer now is responsive.

How I changed it:

I replaced some styles in the CSS file to adjust the responsive design to the footer. For the social media icons, I create a socialMedia.js file to add two functions and open in another window the corresponding social media.
